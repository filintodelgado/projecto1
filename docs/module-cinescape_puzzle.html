PuzzleEventKeyMap

All Puzzle Events available to use:

  • solve is trigged when the .solved property is set to false and the .solve() method is called, normally when the user responds correctly the form;

  • unsolve is trigged when the .solved property is set to true and the .unsolve() method is called;

  • miss is trigged when the .solved property is set to false and the .miss() method is called, normally when the user responds incorrectly the form;

Type:
  • "solve" | "unsolve" | "miss"
