shuffleArray(array)

Shuffles the given array using Fisher-Yates shuffle algorithm.

The algorithm starts by It starts from the last element in the array and swaps it with a randomly selected element from the remaining elements, working its way towards the first element. Finally, it returns the shuffled array.

It modifies the original array.
